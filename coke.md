# コーラシロップのレシピ

## 材料

| 材料                         | 分量  |
| ---                          | ---:  |
| 水                           | 200ml |
| 砂糖                         | 200g  |
| レモン                       | 1個   |
| シナモンスティック           | 2本   |
| バニラビーンズ               | 1/2本 |
| クローブ（ホール）           | 3g    |
| カルダモンシード（ホール）   | 2g    |
| コリアンダーシード（ホール） | 2g    |
| オールスパイス（ホール）     | 2g    |
| コーラナッツ（パウダー）     | 20g   |
| ナツメグ（パウダー）         | 2g    |


## 作り方
1. 全部の材料を鍋に入れて火にかける
2. 沸騰したら弱火にして20分ほど煮る
3. 冷ます
4. ザルで濾す


## 飲み方
炭酸水で三倍くらいに薄める :tropical_drink:
